# Weather app

with react and vite

#### Available Scripts

```node
npm run start
npm run build
npm run serve
npm run codeSpaceStart
npm run test
```

#### package.json

```json
"scripts": {
    "start": "vite",
    "build": "tsc && vite build",
    "serve": "vite preview"
    "codeSpaceStart": "BROWSER=none WDS_SOCKET_PORT=0 react-scripts start",
    "test": "react-scripts start"
  }
```
